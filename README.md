Simple TODO CLI

nothing special, just trying to learn Go

```
Usage:
  todo [command]

  Available Commands:
    add         Add a new todo item
      add [flags]

      Flags:
        -p, --priority int   Priority: 1-5 (default 3)

    list        List todos

  Flags:
    --config string     config file (default is $HOME/.todo.yaml)
    --datafile string   datafile to store todos (default "/Users/dharrigan/.test.json")
    -t, --toggle        Help message for toggle

    Use "todo [command] --help" for more information about a command.:w
```
